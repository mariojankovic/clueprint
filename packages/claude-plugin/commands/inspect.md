Call `mcp__clueprint__inspect` to get the element or region the user selected in the browser.

If the user provided a task with this command, work on that task using the captured context. Otherwise, summarize what was captured and ask what they want to do.

The user selects elements by Option+Click (Mac) or Alt+Click (Windows) in Chrome with the Clueprint extension. They can also Cmd+Shift+Drag to select a region.
